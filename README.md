# Cucinare
Web application that finds recipes from ingredients selected by the user.

#### Technology Stack :
* PHP Scripting Language
* Apache Web Server
* MYSQL Database

#### Tools :
* MySQLWorkbench
* Sublime Text
* manager-osx
